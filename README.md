## 歧语言特点

+ 从 0 设计的中文编程语言。

+ 充分利用现代汉语 (简体中文, `zh_CN`) 的部分语法。

+ 使用简单的语法规则 “解决” 歧义 (遇到歧义则编译错误), 并进行分词。

+ 无空格语法 (除了行首的空格都会被忽略), 参考 Python 的语法设计 (缩进)。

+ 编译为 JavaScript, 具有大部分 JavaScript 的特性, 且能够与 JavaScript 互操作。
